# 如何打开编辑器模式（可编辑）

## 🎯 重要：预览模式 vs 编辑器模式

### 预览模式（只读）
- **打开方式**: 右键 → "Open With" → "WPS Editor"
- **功能**: 只能查看，不能编辑
- **显示**: PDF 或 HTML 渲染

### 编辑器模式（可编辑）
- **打开方式**: 右键 → "Open With" → "WPS Editor (Editor)"
- **功能**: 可以编辑、格式化、保存
- **显示**: 富文本编辑器

---

## 📋 正确打开编辑器的步骤

### 方法 1：使用右键菜单

1. 在 VSCode 资源管理器中找到 .docx 文件
2. **右键点击文件**
3. 选择 **"Open With..."**
4. 在弹出的列表中选择 **"WPS Editor (Editor)"** ⬅️ 注意是 "(Editor)"

```
右键菜单：
├─ WPS Editor                    ← 预览模式（只读）
└─ WPS Editor (Editor)          ← 编辑器模式（可编辑）✅
```

### 方法 2：使用命令面板

1. 选中 .docx 文件
2. 按 **Ctrl+Shift+P** 打开命令面板
3. 输入 **"Open With"**
4. 选择 **"WPS Editor (Editor)"**

### 方法 3：使用命令

1. 按 **Ctrl+Shift+P**
2. 输入 **"WPS Editor: Open Editor"**
3. 回车执行

## 🐛 如果打开的还是预览模式

### 症状：
- 没有工具栏
- 没有格式化按钮
- 不能编辑文字
- 只有简单的查看功能

### 原因：
打开了 "WPS Editor"（预览模式），而不是 "WPS Editor (Editor)"

### 解决：

#### 1. 关闭当前窗口
关闭预览模式的窗口

#### 2. 重新打开
右键文件 → "Open With..." → 选择 **"WPS Editor (Editor)"** ⬅️ 注意 "(Editor)"

#### 3. 如果列表中没有 "(Editor)"
1. 按 **Ctrl+Shift+P**
2. 输入 **"Developer: Reload Window"**
3. 重新加载窗口
4. 再次右键 → "Open With..."

## 🎯 快速测试

### 在编辑器中应该可以：

#### 1. 编辑文字
- 点击任意位置
- 输入文字
- 看到光标闪烁

#### 2. 格式化
- 选中文字
- 点击 **B** → 加粗
- 点击 **I** → 斜体

#### 3. 分页
- 查看分页栏："第 1 页 / 共 X 页"
- 点击 ▶️ 下一页
- 页面滚动

#### 4. 保存
- 点击 **💾 保存** 或按 **Ctrl+S**
- 显示 "文档已保存" 提示

## 🔍 调试：检查打开的是哪个模式

### 在开发者工具 Console 中运行：

```javascript
// 检查是否存在编辑器元素
const editor = document.getElementById('editor');
const toolbar = document.querySelector('.toolbar');
const paginationBar = document.querySelector('.pagination-bar');

console.log({
  '编辑器存在': !!editor,
  '工具栏存在': !!toolbar,
  '分页栏存在': !!paginationBar,
  '模式': (toolbar && paginationBar) ? '编辑器模式 ✅' : '预览模式 ❌'
});
```

**编辑器模式输出**：
```
{
  编辑器存在：true,
  工具栏存在：true,
  分页栏存在：true,
  模式："编辑器模式 ✅"
}
```

**预览模式输出**：
```
{
  编辑器存在：false,
  工具栏存在：false,
  分页栏存在：false,
  模式："预览模式 ❌"
}
```

## 💡 提示

### 默认打开方式
如果直接双击 .docx 文件，VSCode 会使用默认编辑器。建议：
1. 右键 → "Open With..."
2. 选择 "WPS Editor (Editor)"
3. 或点击 "Configure Default Editor..." 设置默认

### 切换模式
- **编辑器 → 预览**: 点击工具栏的 "👁️ 预览" 按钮
- **预览 → 编辑器**: 关闭，重新用 "WPS Editor (Editor)" 打开

### 快捷键
在编辑器模式中：
- **Ctrl+S** - 保存
- **Ctrl+B** - 加粗
- **Ctrl+I** - 斜体
- **Ctrl+U** - 下划线
