# 插件图标设计指南

## 图标要求

### 尺寸和格式
- **推荐尺寸**: 512x512 像素
- **最小尺寸**: 256x256 像素
- **格式**: PNG（支持透明度）
- **文件大小**: < 1MB

### 设计规范

#### 1. 内容建议
- 使用 WPS 或文档相关的图标元素
- 保持简洁、易识别
- 避免过多细节
- 考虑在小尺寸下的显示效果

#### 2. 颜色建议
- 使用与 WPS 品牌相关的颜色（红色、蓝色）
- 确保在亮色和暗色背景下都清晰可见
- 考虑 VSCode 的主题色

#### 3. 设计工具推荐
- [Figma](https://www.figma.com/) - 免费在线设计工具
- [Canva](https://www.canva.com/) - 简易设计工具
- Adobe Photoshop / Illustrator
- [GIMP](https://www.gimp.org/) - 免费开源图像编辑器

## 临时方案

在正式图标设计完成前，可以：

1. **使用占位图标**
   - 使用在线图标生成器
   - 使用 emoji 或简单图形

2. **移除图标配置**
   - 在 package.json 中暂时不配置 icon 字段
   - VSCode 会使用默认图标

## 截图建议

### 推荐截图内容
1. 文档预览界面
2. 编辑模式界面
3. 格式转换功能
4. 设置界面

### 截图规格
- **尺寸**: 至少 1280x720 像素
- **格式**: PNG 或 JPEG
- **数量**: 3-5 张
- **说明**: 每张截图配简短说明

### 截图工具
- macOS: Shift+Cmd+5
- Windows: Win+Shift+S
- [ShareX](https://getsharex.com/) - 免费截图工具
